Documentaion

โปรแกรม store เป็นโปรแกรมที่คล้ายกับร้านค้าออนไลน์ โดยจะต้องทำการสมัครสมาชิกเข้าสู่ระบบเสียก่อนโดยผู้ใช้งานจะต้องทำการฝากเงินเพื่อซิ้อของ และสามารถถอนเงินที่ฝากไว้ออกมาได้อีกด้วย เมื่อซื้อของตามใจอยากแล้วจะคำนวณราคาทั้งหมดและแน่นอนต้องมี VAT7% ยังไงก็ต้องมีนะครับไม่มีไม่ได้ หากจ่ายเงินเสร็จแล้วระบบจะหักเงินออกจากบัญชีของคุณอัตโนมัติ

วิธีการใช้งานไม่ยากเลยใช่ไหมครับ แน่นอนว่าตัวโปรแกรมนี้ยังสามารถนำไปพัฒนาต่อได้อีก แล้วโปรแกรมนี้ยังเป็นแนวทางของคนที่ต้องการศึกษาภาษาซีได้อีกด้วย เพราะโปรแกรมตัวนี้เกิดขึ้นจากการทำงานของภาษาซี สำหรับผู้ที่สนใจในการเขียนโปรแกรมนอกจากจะได้ซื้อของแล้วยังสามารถดูโครงสร้างของ code ผ่านตัวโปรแกรมได้อีกด้วย ส่วนวิธีใช้งานโดยระเอียดแบบเห็นภาพจะอยู่ในหัวข้อ How to use นะครับ

How to use

โดยเมื่อเปิดโปรแกรม Store มาในหน้าแรกจะให้ทำการสมัครสมาชิก หรือทำการเข้าระบบ

ในหน้าสมัครสมาชิกเมื่อเข้ามาแล้ว จะแสดงรหัสไอดี แล้วจะให้กรอกชื่อ นามสกุล username password

เมื่อกดปุ่มอะไรก็ตามจะเข้ามาสู่หน้า เข้าระบบ ให้กรอก username และ password ให้เรียบร้อย

เมื่อเข้าระบบสำเร็จแล้วจะทำการไปหน้า เมนู และให้เลือกว่าจะใช้ระบบไหน ในกรณีนี้จะกด 2 แล้วไปหน้า Banking

กด 1 เพื่อฝากเงินกด 2 เพื่อถอนเงินหลังจากนั้นให้ระบุจำนวนเงิน

ผลลัพธ์เมื่อทำรายการเสร็จสิ้น

การถอนเงินจะทำให้จำนวนเงินนั้นลดลง

หน้า shopping เมื่อเข้ามาจะใก้หรอกเลขของสินค้าที่จะซื้อ แล้วจะให้กรอกจำนวน เมื่อกรอกเสร็จระบบจะถามว่า จะซื้อเพิ่มหรือ คิดเงิน

เมื่อกดคิดเงินจะเด้งเข้ามาหน้าใบเสร็จ มีรายการสินค้าที่ซื้อ ราคารวม + ราคารวม vat7%
